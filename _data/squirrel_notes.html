Squirel is currently by invite only, alpha software at this time.
Encryption looks to be available in the Secret Rooms only.
The app looks like a clone of Discord, very similar setup with groups, rooms, groups chats, direct messages. Currently trying to create a new Direct Message brings you to a blank screen and there does not seem to be any way to add or search for other Squirrel users.
Signup requires that you use the phone number of the phone you are using, trying a different alternate phone number fails. Signup requires you to enter your first and last name and birthdate (though these are not verified).

My verdict: pass
Early software, encryption not the defualt, Yahoo history of data leaks. So far no real information about security features in place.
