A self-hosted chat app for small teams built by Security Compass

Persistent messages - Private and password-protected rooms
Multiple rooms
New message alerts / notifications - Mentions (hey @you/@all)
Image embeds / Giphy search - Code pasting
Let's Chat uses Gravatar to display user avatar images. Each user will need to sign up to Gravatar if they want a pretty picture to appear next to their name in the app.
