Messages are stored in the discord cloud
It is intended to replace Teamspeak and Skype. The messages are not encrypted, there is only a transfer via HTTPS.
My verdict: Stay Away!
Maybe it's useful for gamers who don't necessarily care about privacy. There is no password or other security precautions.
Also the desktop clients are written in JavaScript which is easily modified to bypass security checks.
Did they fix these issues? Maybe but without open code to inspect we cannot know.
